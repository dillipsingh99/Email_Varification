User Authentication With Email.

After creating user account, Django backend send email varification link to user email this only happen if you have done some changes in django settings.py file and also in gmail account, For this you must enable 2-step varification security and create 16-digits apps passwrod and place these stuff in settings.py file then only it works as

EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'

EMAIL_HOST = 'smtp.gmail.com'

EMAIL_HOST_USER = 'your gmail account.'

EMAIL_HOST_PASSWORD = '16-digits apps password'

EMAIL_PORT = 587

EMAIL_USE_TLS = True

DEFAULT_FROM_EMAIL = 'default from email'
